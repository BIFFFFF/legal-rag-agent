# 借条草稿 Skill

## 适用场景

用户要求起草借条、借款凭证或借条模板。

典型请求：

- “帮我写个借条”
- “生成一份借条草稿”
- “借款 5 万，期限 6 个月，帮我写借条”

## 需要确认的信息

- 出借人姓名、身份证号、联系方式。
- 借款人姓名、身份证号、联系方式。
- 借款金额。
- 借款用途。
- 借款交付方式。
- 借款期限。
- 利率和逾期责任。
- 还款方式。
- 是否有担保人。
- 争议解决方式。

## 生成步骤

1. 根据用户已提供信息填入草稿。
2. 信息不足时保留占位符。
3. 提醒用户保留转账凭证。
4. 提醒用户利率不得超过法律保护上限。
5. 保存 Markdown 草稿。

## 建议结构

```markdown
# 借条

今借到【出借人姓名】【身份证号】人民币【金额】元整（￥【金额数字】），用于【借款用途】。

借款人确认上述款项已通过【现金/银行转账/微信/支付宝】方式实际收到。

借款期限自【起始日期】至【到期日期】。

双方约定借款利率为【利率】。借款人应于到期日前一次性归还本金及利息。

借款人逾期未还的，应承担逾期还款责任。

如发生争议，双方同意由【法院名称】管辖。

借款人：
身份证号：
联系方式：

出借人：
身份证号：
联系方式：

签署日期：
```

## 风险提醒

- 尽量通过转账交付款项，并备注“借款”。
- 补充身份证号和联系方式，避免主体不清。
- 大额借款建议保留聊天记录、催款记录和还款记录。
