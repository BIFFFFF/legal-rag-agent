# 解除合同通知书草稿 Skill

## 适用场景

用户要求通知对方解除合同，或希望起草解除合同通知书。

典型请求：

- “帮我写解除合同通知”
- “对方违约了，我想解除合同”
- “生成一份解除合同通知书”

## 需要确认的信息

- 通知人和被通知人。
- 合同名称和签订日期。
- 合同主要内容。
- 对方违约事实或解除原因。
- 合同约定的解除条款。
- 法律规定的解除依据。
- 通知送达方式。
- 后续返还、结算、赔偿等要求。

## 建议结构

```markdown
# 解除合同通知书

## 一、通知对象

## 二、合同基本情况

## 三、解除原因

## 四、解除依据

## 五、后续处理要求

## 六、通知送达与生效

## 七、通知人信息
```

## 写作要求

- 不要在依据不足时直接断言一定可以解除合同。
- 应提示用户确认是否具备合同约定或法律规定的解除条件。
- 后续处理要求要具体，例如返还款项、交还资料、停止履行、赔偿损失。
